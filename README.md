# suitability writer v3.11

Minimal starter workspace for a Python-based suitability writer project.

## Requirements

- Python 3.11 or newer

## Run

Run the starter script:

```powershell
python main.py
```

## Debug

Open `main.py` in VS Code and run the Python debugger from the Run and Debug view.

## Files

- `main.py` - starter entry point
- `.github/copilot-instructions.md` - workspace setup checklist and notes
