# Golden Case – Suitability Writer v3.x

This Golden Case demonstrates a complete, regulator-grade advice lifecycle.

## Purpose
- Provide a canonical end-to-end reference case
- Demonstrate control enforcement and evidential integrity
- Enable independent reconstruction of advice output

## Scope
- One real advice case
- One system version
- One deployment

## Guarantees
- All artefacts are immutable
- All hashes can be recomputed
- Output can be reconstructed from inputs + approved artefacts

Any change to schemas, templates, or validation logic requires a new Golden Case.
