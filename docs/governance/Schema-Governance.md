# Schema Governance

_Authoritative documentation for the evidence system._

## Canonical encrypted envelope schema

The authoritative JSON Schema for the encrypted evidence bundle envelope is:

- `https://schemas.consilium.co.uk/evidence/encrypted-bundle-envelope/1.0.0.json`

All runtime, test, and verification references for the evidence envelope should align to schema version `1.0.0` unless a later version is formally approved and published through governance.
